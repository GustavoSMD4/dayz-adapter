Documentação da API

GET /files/names

Retorna uma lista com os nomes dos arquivos disponíveis.

Resposta: 
[
  "arquivo1",
  "arquivo2",
  "arquivo3"
]

GET /file/get/<name>

Retorna o conteúdo do arquivo name codificado em base64 (texto plano).

Parâmetros:
Parâmetro Descrição
name Nome do arquivo a ser obtido
Resposta: 
Base64 encoded string of the file content

GET /file/get/<name>/json

Retorna o conteúdo do arquivo name formatado como JSON.

Parâmetros:
Parâmetro Descrição
name Nome do arquivo a ser obtido
Resposta: 
JSON formatado do arquivo

POST /generator

Gera um arquivo JSON base no tipo e coordenadas passadas.

Request JSON: 
{
  "type": "tent",
  "coordinates": [10.0, 5.0, 3.0]
}
Resposta: 
Base64 encoded JSON string

POST /file/upload

Faz upload de um arquivo JSON codificado em base64 com uma descrição.

Request JSON: 
{
  "descricao": "Descrição do arquivo",
  "file": "base64_encoded_file_content"
}
Resposta: 
"Saved"

POST /file/delete

Deleta um arquivo pelo nome.

Request JSON: 
{
  "descricao": "Nome do arquivo a deletar"
}
Resposta: 
"Deleted"
